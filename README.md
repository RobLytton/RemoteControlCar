# RemoteControlCar
This project has 2 arduinos, one is on a 2 wheel car (acts as receiver) while the other acts as the controller. 

The controller has 2 joysticks one for each wheel, 4 buttons and a light sensor.

The  car can travel in all directions, has 2 ultrasonic sensors, a buzzer, and dark activated headlights

The 3 of buttons on the controller are for safety mode, the horn (buzzer), and the 3rd autonomuos mode. 
The light sensor turns on the headlights in dark enviornments
